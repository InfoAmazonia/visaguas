# visaguas

Uma análise sobre a água na Amazônia Legal

## Pré-requisitos

 - npm 1.4.x
 - node 0.10.x

## Instalação

Clone o repositório

```
$ git clone https://github.com/oeco/visaguas.git
```

Instale a aplicação digitando:

```
$ cd visaguas
$ npm install
```

Inicie:

```
$ npm start
```

Para visualizar, acesse [http://127.0.0.1:8000](http://127.0.0.1:8000)
